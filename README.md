## ğŸ§® Data Preparation & Modeling

Data from the four Excel sources were **cleaned, standardized, and integrated** in Power BI using Power Query.
Each dataset was transformed to ensure consistency in fields such as **date formats**, **store identifiers**, **product categories**, and **revenue columns**.

Key steps included:
- Removing duplicates and null entries.
- Normalizing product and region naming conventions.
- Creating relational links between datasets using **store and product IDs**.
- Building calculated measures in DAX for metrics such as:
  - *Total Sales*
  - *Profit Margin*
  - *Revenue Growth Rate*
  - *Average Order Value (AOV)*

This foundation ensured reliable and efficient data modeling for accurate dashboard performance.
